# Gindex
Google Indexing Using Indexing Api Python 
Submit URLs in bulk to Google Indexing, fetching automatic urls from Sitemap.xml
```
git clone https://github.com/DomathID/Gindex
cd Gindex
pip install -r requirements.txt
python run.py
```
